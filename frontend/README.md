# AI Beauty Platform - Frontend

Frontend приложение для AI Beauty Platform на React + TypeScript + Vite.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

## Сборка

```bash
npm run build
```

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Zustand
- TanStack Query
- React Hook Form + Zod
